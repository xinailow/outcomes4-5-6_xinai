2. Explain user-agent.

Answer:

The user agent is a browser text string that is given to each website we want to visit. This string indicates the browser you are using, its version number, and the details of the system. Using this data, the site can evaluate your computer to optimize page performance and display.

Answer:

The factors that affect the file size of an image is file compression. The role of compression technology is that most of the data in most cases is redundant. Redundancy can be explained by a large number of background pixels with the same or similar color. By lossless decompression, the data is the same as the original data. Using decompression, some data is removed and decompressed, can changes the file size different from the original.

You can apply a real shadow to disappear in the background. You can create any shape or form of image. Alpha transparency has many possibilities.
